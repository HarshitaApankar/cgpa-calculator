# CGPA Calculator

A free, simple web tool for students to calculate their semester GPA and cumulative CGPA instantly.

## 🔗 Live Demo
https://cgpa-calculator-nine-beta.vercel.app

## ✨ Features
- Add multiple semesters and subjects
- Supports 10-point (Indian), 4-point (US), and Percentage grading scales
- Calculates semester GPA and cumulative CGPA in real time
- No signup, no data stored, works offline
- Fully responsive — works on mobile and desktop

## 🛠️ Built With
- HTML, CSS, JavaScript (vanilla)
- Deployed on Vercel (free)

## 🚀 How to Use
1. Select your grading scale (10-point / 4-point / Percentage)
2. Click **+ Add Semester**
3. Add subjects with their credits and grades
4. CGPA updates automatically!
